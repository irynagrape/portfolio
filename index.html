Pocket Ledger

Pocket Ledger is a lightweight personal finance web app built with vanilla HTML, CSS, and JavaScript. It lets users add and manage income and expense transactions, organize categories, filter and search records, and track monthly spending trends through dashboard charts. The app runs fully in the browser with LocalStorage, so data persists locally without any backend.
